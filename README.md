## Install

```sh
npm install
npm start:dev
```

## Environment

Ejemplo:
```sh
# APP
PORT=3000

# DATABASE
MYSQL_HOST=localhost
MYSQL_PORT=3306
MYSQL_DATABASE=database
MYSQL_USER=root
MYSQL_PASSWORD=root
MYSQL_CONNECTION_LIMIT=10
```
